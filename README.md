# Prettier config

## Usage

Install package:

```bash
yarn add -D @fethcat/prettier-config
```

Install prettier:

```bash
yarn add -D prettier
```

Add it to your package.json file:

```json
{
  "name": "@fethcat/version",
  "devDependencies": {
    "@fethcat/prettier-config": "^x.x.x",
    "prettier": "^x.x.x"
  },
  "prettier": "@fethcat/prettier-config"
}
```
